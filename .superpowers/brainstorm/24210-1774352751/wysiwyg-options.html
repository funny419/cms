WYSIWYG 에디터 라이브러리

A 레이아웃(풀페이지)에 적용할 에디터를 선택해주세요

B I U | H1 H2 | ≡ 목록 1. 번호 | 🔗 🖼

여기에 내용을 직접 서식 있게 작성합니다.

텍스트를 드래그하면 툴바가 나타납니다.

A — Quill (react-quill)

가장 널리 쓰이는 React WYSIWYG. 상단 고정 툴바, 안정적. npm 설치만으로 사용 가능.

설치: npm install react-quill

제목 텍스트

텍스트를 선택하면 B|I|🔗|H1 형태의 팝업 툴바 등장.

/를 입력하면 블록 삽입 메뉴...

텍스트 선택 시 플로팅 툴바 · / 입력 시 블록 메뉴 (Notion 스타일)

B — TipTap

Notion과 동일한 ProseMirror 기반. 선택 시 플로팅 툴바, / 슬래시 커맨드. 가장 현대적.

설치: npm install @tiptap/react @tiptap/starter-kit

| B I U | ≡ 1. | 🔗 🖼 표

풍부한 기능의 에디터. 표, 미디어 삽입, 고급 서식 지원.

C — CKEditor 5

기능이 가장 풍부함. 표·미디어 삽입 등 고급 기능. 번들 크기가 크고 설정이 복잡.

설치: npm install @ckeditor/ckeditor5-react
